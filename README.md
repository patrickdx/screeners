1. **GitHub Actions** runs the workflow daily at 9:00 AM UTC

4. GitHub Pages serves `index.html` which fetches and displays the data

## â• Adding a New Screener

To add "Screener C":

1. **Create the Python script**: `scripts/screener_C.py`
   ```python
   import json
   import os
   
   stock_data = [{"ticker": "EXAMPLE", "metric": 123}]
   os.makedirs("data", exist_ok=True)
   with open("data/screener_C.json", 'w') as f:
       json.dump({"data": stock_data}, f, indent=2)
   ```

2. **Update the workflow**: Add to `.github/workflows/run_screeners.yml`
   ```yaml
   - name: Run Python Screeners
     run: |
       python scripts/screener_A.py
       python scripts/screener_B.py
       python scripts/screener_C.py  # Add this line
   ```

3. **Update the frontend**: Add to `index.html`
   ```html
   <!-- Add button -->
   <button id="loadScreenerC">Load Screener C</button>
   
   <!-- Add config -->
   screenerC: {
       file: "data/screener_C.json",
       columns: [
           { "title": "Ticker", "data": "ticker" },
           { "title": "Metric", "data": "metric" }
       ]
   }
   
   <!-- Add handler -->
   $('#loadScreenerC').on('click', function() {
       loadScreenerData(tableConfig.screenerC);
   });
   ```
